Калькулятор падения напряжения по длине кабеля

Расчёт для линии с последовательно расположенными нагрузками. Для каждого участка учитывается ток всех последующих потребителей.

Общие параметры

Нагрузки по линии

Каждый участок — это расстояние от предыдущей точки до следующего потребителя.

Пример:
Источник → 10 м → Нагрузка 1 → 7 м → Нагрузка 2 → 5 м → Нагрузка 3

Для участка 10 м считается ток нагрузок 1+2+3.
Для участка 7 м считается ток нагрузок 2+3.
Для участка 5 м считается ток нагрузки 3.
